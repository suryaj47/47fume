# 🌸 About

**47FUME** is a premium luxury perfume landing page inspired by modern luxury brands.

The goal was to create a cinematic browsing experience where every section—from typography to animations—reflects elegance and sophistication.

Instead of relying on heavy frameworks, the project demonstrates how **HTML, CSS, and JavaScript** alone can produce an immersive, responsive, and visually polished website.
